### A/B Testing Ideas

**Subject Lines**:
- Question vs statement
- Emoji vs no emoji
- Personalization vs generic
- Urgency vs curiosity

**Send Times**:
- Tuesday 10am vs Thursday 2pm
- Weekend vs weekday
- Morning vs evening

**Email Length**:
- Short (200 words) vs long (600 words)
- Scannable (bullets) vs prose

**CTA Placement**:
- Top vs bottom vs both
- Button vs link vs text

### Email Copywriting Rules

1. **Subject line** (<50 characters, curiosity + benefit)
2. **Opening** (personal, relevant, hook in first line)
3. **Body** (one main idea, easy to scan, conversational)
4. **CTA** (clear, specific, easy to find)
5. **Closing** (sign off with name, keep it human)
6. **P.S.** (use it! 90% of people read it)

### Technical Setup Checklist

- [ ] SPF/DKIM/DMARC configured (avoid spam)
- [ ] Custom sending domain (not @gmail.com)
- [ ] Double opt-in enabled (cleaner list)
- [ ] Unsubscribe link in every email (legal requirement)
- [ ] Mobile-responsive templates (60%+ open on mobile)
- [ ] Plain text versions (for email clients that block HTML)
- [ ] Tracking pixels (opens) and link tracking (clicks)
- [ ] Bounce handling (remove invalid emails)
